# LeanMe Site

Sito ufficiale LeanMe Srls — Aziende Ibride.

## Stack

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS 4
- Framer Motion
- Geist Font
- Vercel deployment

## Sviluppo locale

```bash
npm install
npm run dev
```

Apri [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Documentazione

La documentazione di progetto è in `docs/` e costituisce la fonte di verità.

## Placeholder

Le immagini placeholder sono chiaramente marcate con `data-replace-hint` e possono essere sostituite aggiornando i file in `data/` e `public/images/`.
